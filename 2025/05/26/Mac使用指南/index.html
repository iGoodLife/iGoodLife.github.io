Mac使用指南

顶部导航栏

Mac图标

系统偏好设置 => Windows控制面板

导航栏剩余部分

显示当前正在使用软件的功能菜单。例如打开Xmind。

状态栏 => Windows右下角

按住command＋鼠标左键，可以拖动图标改变图标位置，甚至删除图标。

底部导航栏

左边是程序，右边是文件。

程序又被小黑线分为两部分，左边是常驻应用程序；右边是最近使用过但未常驻，这部分程序在启动台中(左2)。

文件部分包含 文件、垃圾桶，最小化程序时(比如最小化网易云)也会暂放在文件部分。

触控板

光标与点按

滚动缩放

更多手势

四指向上是展示 所有应用和桌面的窗口；而向下是展示 当前应用的所有窗口。

  • 单击：按下触摸板才能单击，

​ 修改方式：Apple -> 系统偏好设置 -> 触控板 -> 光标与点按 -> 勾选 轻点来点按。

  • 三指拖移：框选、拖移文件

    修改方式：Apple -> 系统偏好设置 -> 辅助功能 -> 指针控制 -> 触碰版选项 -> 勾选 启用拖移。

​ Ps：貌似四指改三指也是在这里？

键盘

command代替了ctrl, option代替了alt。

而control一般用来和 空格使用 切换输入法，或者一些配合复杂的快捷键。

复制

command + d，等价于 Windows中的 复制＋粘贴 同时完成。

拷贝

command + c，等价于 Windows中的 复制。

粘贴

command + v，等价于 Windows中的 粘贴。

剪切

剪切command + x分为两部分，针对文本 和 针对文件。

  • 文本：同理Windows，剪切掉文字再粘贴即可。
  • 文件：要先 拷贝，再 command+option+v，从而实现Windows中的效果，原位置的文件转移到了新位置。

鼠标

上下滚动

修改方式：Apple -> 系统偏好设置 -> 鼠标 -> 取消勾选 滚动方向-自然；跟踪速度 拉满。

取消勾选后，触摸板的方向貌似也反过来了？有一个叫Mos的插件，可以同时让两个变得舒服~

实际操作

神器1：文件预览

单击选中文件，按空格即可预览。

神器2：聚焦搜索

右上角放大镜图标，快捷键：command+空格

不仅仅可以查找文件，还能翻译、计算机、汇率换算等等。

截图/录制

全屏截图：shift+command+3

框选截图：shift+command+4

shift+command+5 是截图/录制功能栏，包括上述两种截图方式以及录屏。

全屏分屏

Mac左上角是Windows右上角的缩小、放大、关闭。

在全屏应用程序后，可以四指左右滑动来切换全屏程序。

如果想要分屏，先四指上推再三指将两个程序拖动到一起。

关闭程序

点左上角的 ×，只是放到了后台运行。

彻底关闭的正确做法：1. command+q 2. 左上角菜单栏关闭 3. 程序坞右键关闭

强制退出

类似windows中的任务管理器来结束程序。

点击Apple图标 -> 强制退出。

或者打开 活动监视器，点击要关闭软件，再点 ×。

活动监视器类似于 任务管理器，可以看内存、CPU占用等。

程序安装

  • AppStore 类似 iPad

  • 打开软件官网，下载Mac版本，得到.DMG文件

    流程：打开DMG文件，将左侧的应用程序 拖动到 右侧的Applications文件夹中，最后在最左侧位置处将应用程序临时挂载的安装磁盘推出。打开启动台即可看到新安装的应用程序。

  • 突破限制的安装软件！
    Apple -> 系统偏好设置 -> 安全与隐私 -> 通用 -> 单击左下角的锁按钮 -> 任何来源。

​ 如果没显示任何来源，打开 启动器 -> 其他 -> 终端，输入以下命令和开机密码。(这里输入密码不显示

​

文件管理

访达 -> 偏好设置 -> 边栏，勾选需要常驻在左侧的内容。
